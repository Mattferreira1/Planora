## ✅ Checklist de Setup

Use a [DOCUMENTACAO_TECNICA.md](DOCUMENTACAO_TECNICA.md#-setup-e-instalação) como referência:

```bash
□ Node.js 18+ instalado
□ PostgreSQL 12+ instalado
□ Google Cloud account criado
□ Clonar repositório
□ npm install
□ Configurar .env.local
□ npx prisma migrate dev
□ npm run dev
□ Acessar http://localhost:3000
```
